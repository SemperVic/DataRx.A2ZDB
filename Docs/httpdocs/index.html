Introducing IDEF1X

Integration DEFinition for information modeling (IDEF1X) is a data modeling language for the development of semantic data models. IDEF1X is used to produce a graphical information model which represents the structure and semantics of information within an environment or system.

Data Storage Options

A2ZDB supports both relational and/or dimensional storage strategies.

  • MS SQL, MS SQL Express, SQLite 3, PostgreSQL, MySQL
  • Mongo DB, CouchDB

Taxonomy Namespace

The A2Z data model is first and foremost designed from an enterprise perspective that is a holistic top-down logical model. What this means is the model is designed to satisfy identifiable and predictable "Real World" modelling patterns.
