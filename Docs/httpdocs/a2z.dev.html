DEV: Development Operations Object Model

The DevOps taxonomy schema is dedicated to development operations. Core functionality includes a light weight project/task management model, a project documentation model as well as a calendar and a standardized object notation model. The DevOps taxonomy schema does have a dependency on the SDK taxonomy schema when generating code and tracking change management through the DevOps project toolsets.
