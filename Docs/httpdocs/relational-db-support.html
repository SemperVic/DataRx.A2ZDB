A2Z Relational Database Support

This documentation provides useful information about the relational databases A2ZDB supports

Database Support

  • SQLite 3
  • PostgreSQL 10.3
  • MSSQL 2012
  • MSSQL Express
  • MySQL

Abstract Data Factory Pattern

Arizona's runtime, persists data via a service layer definition that can be configured to implement the database platform you prefer by defining the DBFactory option. If no factory is declared, Arizona will assume default connections to an internal SQLite3 database.

Call Stack Architecture

The A2Z data model is first and foremost designed from an enterprise perspective that is a holistic top-down logical model. What this means is the model is designed to satisfy identifiable and predictable "Real World" modelling patterns.
