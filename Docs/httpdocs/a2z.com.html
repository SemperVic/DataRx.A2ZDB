COM: Community Object Model

The Community Object Model is a core taxonomy schema as this is where member's of the community are identified, archetyped and extended. The Community Object Model is the who's who of A2ZDB taxonomy. For an in-depth review of this model, please refer to the article titled "Anatomy of a Taxonomy Schema". Like metadata, Member data is often exdended into just about every other taxonomy schema A2ZDB supports.
